Privacy Policy for FIFA World Cup 2026 Match Center

Effective date: June 14, 2026

FIFA World Cup 2026 Match Center is a Chrome extension that provides a compact match-center popup for FIFA World Cup 2026 scores, fixtures, group standings, completed results, and road-to-final information.

Information We Collect

This extension does not collect personal information from users.

We do not collect names, email addresses, phone numbers, payment information, authentication information, health information, personal communications, browsing history, website content, keystrokes, or user activity.

How the Extension Works

The extension fetches publicly available sports data from ESPN API endpoints to display match schedules, live scores, match results, group standings, and knockout fixtures.

The extension also loads country flag images from flag image providers to display team and host-country flags inside the popup.

Local Storage

The extension may save a simple theme preference locally in the browser, such as dark mode or light mode. This setting stays on the user’s device. It is not transmitted to us.

Third-Party Services

The extension may request sports data or flag images from third-party domains, including ESPN API domains and flag image providers. These requests are used only to display tournament information in the extension.

We do not sell user data. We do not use user data for advertising. We do not use user data for creditworthiness or lending decisions.

Remote Code

The extension does not use remote code. All JavaScript, HTML, and CSS are included inside the extension package. The extension does not load external scripts, external WebAssembly, external modules, or executable code from remote servers.

Data Sharing

We do not sell, rent, or share personal user data with third parties.

Data Security

The extension uses HTTPS endpoints where available for sports data and image requests.

Children’s Privacy

The extension is not designed to collect information from children or any other users.

Unofficial Status

This is an unofficial fan-made extension. It is not affiliated with, endorsed by, or sponsored by FIFA, ESPN, or any official tournament organization.
